The babynames dataset in the babynames packages contains all births in the U.S. from 1880 to 2015 in which there were at least 5 births of that sex-name dyad in that year. (A “dyad” is a combination of two variables, “triad” is three, etc.)

Create a dataset called babynames_total that totals births across all years. (Note: The case in the new dataset should be sex and name, which means that each of these dyads should appear only once in the dataset.)

Use this dataset to calculate what is asked below:

library(tidyverse)
library(babynames)
babynames_total <- babynames %>% 
  group_by(sex, name) %>% 
  summarize(n = sum(n), .groups="drop")

  3. Based on the way the data were assembled, explain why there are so many.

    There are so many sex-name pairs with 5 births because, as mentioned in the description of the babynames data set at the beginning of the assignment, the data set only includes births “in which there were at least 5 births of that sex-name dyad in that year.” For this reason, the smallest value of births possible is 5. The reason there are so many can simply be explained by the presence of unique, uncommon names that may have been used one year and then disappeared the next.
